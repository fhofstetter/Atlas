# Output File Rules

- Every file in `output/` must begin with a comment line identifying the
  generating agent and timestamp:
  `<!-- Generated by <agent-name> agent — <ISO timestamp> -->`
- Filenames follow: `<task-id>_result.md` for task results, or
  `<workflow-name>_<step>_result.md` for workflow step outputs.
- Never write credentials, secrets, or `.env` values into output files.
